# Personalized Cover Letter Generator
This project generates personalized cover letter using AI based on user's resume and job description.
## Technologies Used
-Python
-Streamlit
-Ollama
-PaddleOCR
## Features
-Extracts text from resume using OCR
-Generates AI-based cover letters
-Simple web interface using Streamlit
## Install Ollama
 -run the ollama using  ollama run llama3
## Run the Project
pip install -r requirements.txt
 run the application using Streamlit run app.py
